# Paper Assistant Lite v3.4.0

> 面向国内硕士研究生的全流程学术写作 AI Skill，基于 WorkBuddy 平台。

---

## 这个 Skill 是干什么的？

当你第三次告诉 AI "这段读起来像机器写的，帮我改自然一点"，却拿到一份只是把"显著提升"换成"有了明显进步"的换词稿时——  
问题不在你，在于 AI 缺乏一个真正懂学术论文的"内部审稿人"。

**Paper Assistant Lite 把学术写作中经过验证的方法论、模板和质量检查机制，打包成一个 AI 可以稳定执行的技能包。**  
让你不用在 prompt 调试上浪费时间，把精力留给真正的思考和写作。

---

## 核心能力

| 模块 | 功能 |
|------|------|
| 📚 深度文献研究 | 系统综述/轻量综述双模式、中文数据库增强检索（知网/万方/维普/百度学术）、PRISMA 2020、GRADE/ROBIS/CASP |
| ✍️ 论文写作 | 学位论文/期刊论文/开题报告三模式、国内硕士论文完整结构（七章制）、各章写作指导+检查清单、问卷编制/访谈提纲/观察记录设计 |
| 🔍 论文评审 | EIC+方法学+领域+统计+反方 五角色评审、AIGC 风险审查、报告规范合规检查（CONSORT/PRISMA/STROBE 等）、0-100 评分 |
| 📊 学术图表 | 7 大类型（描述统计表/相关矩阵/条形图/箱线图/散点图/三线表/信效度表）+ Python 即用代码 + 论文图表质量标准 |
| 🛡️ 引用零伪造 | 所有文献必须先搜索验证入池再使用，文献池跨会话持久化 |
| 🤖 降低 AIGC 率 | 七规则：句长落差/引述多样化/段落反对称/研究者声音/领域用语/具体化/不完美允许 |

## 快速开始

| 场景 | 你直接说 |
|------|---------|
| 写硕士论文 | 帮我写关于[主题]的硕士论文，学校是[校名] |
| 写开题报告 | 帮我写关于[主题]的开题报告 |
| 做文献综述 | 帮我做[主题]的文献综述，硕士论文用 |
| 润色/去AI味 | 这段读起来像AI写的，帮我改自然一点 |
| 评审论文 | 帮我评审这篇论文：[粘贴/上传] |
