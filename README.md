## 为什么需要这个 Skill？

> 当你第三次告诉 AI "这段读起来像机器写的，帮我改自然一点"，却拿到一份只是把"显著提升"换成"有了明显进步"的换词稿时——问题不在你，在于 AI 缺乏一个真正懂学术论文的"内部审稿人"。

学术写作的 prompt 工程正在成为一种**隐性资源**：有经验的研究者有一套自己的润色模板和自检清单，而大多数人还在从零摸索、反复调教。

这个 Skill 的目标很简单——**让你不用在 prompt 调试上浪费时间，把精力留给真正的思考和写作。**

## 核心哲学：引用来源锁定

```
搜索 → 验证 → 加入文献池 → 从池中引用
```

**铁律**：

- 所有引用必须先搜索验证，零伪造
- 用户提供文献仍需验证
- 搜索不到 → 不入论文，不创建"待验证"标记
- 文献池跨会话持久化，长论文写作不丢文献

## 它和其他方案有什么不同？

| 维度 | 通用 AI 对话 | 本 Skill |
|------|-------------|----------|
| **文献引用** | 可能编造文献 | 强制搜索→验证→入池→引用，零伪造 |
| **写作结构** | 通用模板 | 国内硕士论文七章制，逐章指导 + 检查清单 |
| **中文适配** | 偏英文学术习惯 | 知网/万方/维普检索路径、GB/T 7714 引用格式 |
| **AIGC 防护** | 无 | 七规则全链路 + 六维自检 Execution Protocol |
| **研究工具** | 给出笼统建议 | 量表编制六步法、信效度检验方案、伦理材料模板 |
| **论文评审** | 泛泛而谈 | 五角色评审 + CONSORT/PRISMA/STROBE 规范检查 + 0-100 评分 |
| **数据分析** | 直接出结果 | 前提假设检验→方法选择→效应量→置信区间→图表，完整流程 |

## 优先级遵循体系

本 Skill 内容较长，AI 在上下文有限时可能无法同时遵循所有指令。以下为最低遵循清单：

| 优先级 | 规则 | 违反后果 |
|--------|------|---------|
| **P0** | 引用来源锁定，零伪造 | 论文废掉 |
| **P0** | 正文基于真实文献，不编造 | 论文废掉 |
| **P1** | 统计方法先检验前提假设 | 方法选错结论无效 |
| **P1** | 研究工具必须有信效度说明 | 方法学评审不通过 |
| **P2** | AIGC 七规则 | AIGC 检测率偏高 |
| **P2** | 三线表 + 效应量 + 置信区间 | 格式不规范 |
| **P3** | Role 三层构造 + 修改阈值 | 文本质量下降 |
| **P3** | 反方检验 + 替代解释 | 论证深度不足 |

> 使用建议：新对话开始时告知 AI "请遵循 P0 和 P1 规则"即可启动。上下文紧张时放弃 P3，优先保住 P0-P1。

## 谁适合用？

- 🎓 国内硕士研究生（教育学、心理学、社会学及所有人文社科）
- 📝 需要写开题报告、学位论文、期刊论文的研究者
- 🔬 需要做文献综述（轻量 / PRISMA 系统综述）的研究生
- 🧑‍🏫 需要设计研究工具（问卷/访谈/观察）的实证研究者
- 📊 需要做统计分析 + 生成出版级图表的教育/社科研究者

> 示例以教育学为主，方法论适用于所有人文社科及部分理工科领域。
