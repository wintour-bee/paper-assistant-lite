# 中文学术论文写作规范速查 v3.2

> 配套 paper-assistant-lite v3.2 使用。核心哲学：引用来源锁定——先搜索、验证、入池，再引用。

## GB/T 7714-2015 参考文献格式

### 期刊论文 [J]
```
作者. 题名[J]. 刊名, 年, 卷(期): 起止页码.

示例：
张华, 李萍. 融合教育环境下自闭症儿童学习动机研究[J]. 中国特殊教育, 2023, 30(4): 45-52.
```

### 专著 [M]
```
作者. 书名[M]. 出版地: 出版社, 年.

示例：
陈向明. 质的研究方法与社会科学研究[M]. 北京: 教育科学出版社, 2000.
```

### 专著中析出文献 [M]//
```
析出文献作者. 析出文献题名[M]// 专著作者. 专著题名. 出版地: 出版社, 年: 析出文献起止页码.

示例：
刘良华. 行动研究的理论与方法[M]// 丁钢. 中国教育研究的新进展. 上海: 华东师范大学出版社, 2015: 120-135.
```

### 学位论文 [D]
```
作者. 题名[D]. 保存地点: 保存单位, 年.

示例：
王芳. 随班就读自闭症儿童社会交往干预研究[D]. 上海: 华东师范大学, 2022.
```

### 会议论文集 [C]
```
作者. 题名[C]// 会议论文集编者. 会议论文集名. 出版地: 出版社, 年: 起止页码.

示例：
李明, 赵强. AI辅助教学在特殊教育中的应用探索[C]// 中国教育技术协会. 2024教育技术国际研讨会论文集. 北京: 高等教育出版社, 2024: 89-95.
```

### 报纸文章 [N]
```
作者. 题名[N]. 报纸名, 出版日期(版次).

示例：
周丽. 融合教育政策落地的新挑战[N]. 中国教育报, 2024-03-15(3).
```

### 标准 [S]
```
标准编号 标准名称[S]. 出版地: 出版者, 年.

示例：
GB/T 7714-2015 信息与文献 参考文献著录规则[S]. 北京: 中国标准出版社, 2015.
```

### 科技报告 [R]
```
作者. 题名[R]. 报告题名及编号. 保存地点: 保存单位, 年.

示例：
教育部基础教育司. 随班就读工作年度报告[R]. 北京: 教育部, 2023.
```

### 专利 [P]
```
专利所有者. 专利题名[P]. 专利国别: 专利号, 公告日期.

示例：
张三. 一种智能教学辅助装置[P]. 中国: CN202310123456.7, 2024-01-15.
```

### 电子资源 [EB/OL]
```
作者. 题名[EB/OL]. (发布日期)[引用日期]. 获取和访问路径.

示例：
教育部. 关于加强残疾儿童少年义务教育阶段随班就读工作的指导意见[EB/OL]. (2020-06-22)[2024-05-10]. http://www.moe.gov.cn/srcsite/A06/s3321/202006/t20200628_518651.html.
```

### 引用网络课程/教学视频 [EB/OL]
```
制作者. 视频标题[EB/OL]. (发布日期)[引用日期]. 平台名称及URL.

示例：
中国大学MOOC. 特殊教育学概论[EB/OL]. (2023-09-01)[2024-05-10]. https://www.icourse163.org/course/XXXX.
```

### 英文文献在中文论文中的格式
- 作者名、题名、期刊名均保留原文
- 类型标识后可加语言标识：[J/OL]（英文期刊在线）

### 多作者文献
- 作者不超过3人：全部列出
- 作者超过3人：列出前3人，后加", 等"（中文）或", et al"（英文）
- 示例：张华, 李萍, 王明, 等. 题名[J]. ...

## 中文文献验证路径（P1增强）

### 中文期刊无DOI的常见情况

中文期刊 DOI 注册率较低，尤其2015年以前的文献。验证策略：

```
有DOI → Crossref API 精确匹配 → 最可靠路径
无DOI → WebSearch "<期刊名> <年份> 第<卷>卷 第<期>期"
       → WebFetch 期刊官网/知网/万方页面
       → 核对：标题 + 作者 + 年份 + 卷期 + 页码（五项一致）
       → 验证来源标注："知网" / "万方" / "期刊官网"
```

### 中文数据库验证优先级

| 优先级 | 来源 | 验证能力 | 适用文献 |
|--------|------|---------|---------|
| 1 | 期刊官网 | 卷期页码+全文 | CSSCI/北大核心期刊 |
| 2 | 中国知网 | 元数据+摘要 | 大部分中文学术期刊 |
| 3 | 万方数据 | 元数据+摘要 | 知网未收录的期刊 |
| 4 | 维普 | 元数据 | 补充验证 |
| 5 | 百度学术 | 发现+初步验证 | 非正式/灰色文献 |

### 中文文献入池元数据要求

中文文献入池时，以下字段必须填写：

```
条目ID | 标题 | 作者 | 年份 | 期刊名 | 卷(期): 页码 | DOI(如有) | 验证来源 | 关键发现 | 被引次数
```

- DOI 为空时标注"无DOI"
- 验证来源标注具体平台（知网/万方/期刊官网等）
- 被引次数可从知网/万方/百度学术获取

## 引用搜索入池协议

### 中文文献五层验证流程

```
中文文献 → Layer1: 有DOI? → Crossref API 验证
              ↓ 无DOI或失败
           Layer2: WebSearch "<标题> <作者>" → 找知网/万方页面
              ↓ 失败
           Layer3: WebSearch "<期刊名> <年份> <期号>" → 期刊官网
              ↓ 失败
           Layer4: WebSearch "<标题> + 研究论文/学位论文" → 百度学术
              ↓ 仍然失败
           Layer5: 无法验证 → 不入池 → 告知用户 → 不在论文中引用
```

### 英文文献五层验证流程

```
英文文献 → Layer1: DOI → Crossref API
              ↓ 失败
           Layer2: 标题 → Semantic Scholar API
              ↓ 失败
           Layer3: 标题 → WebSearch
              ↓ 失败
           Layer4: 期刊+卷期+页码 → WebSearch 期刊官网
              ↓ 仍然失败
           Layer5: 无法验证 → 不入池 → 告知用户 → 不在论文中引用
```

### 验证判断标准

| 结果 | 含义 | 处理方式 |
|------|------|---------|
| 验证通过 | 在权威数据库找到匹配 | 正常入池引用 |
| 部分匹配 | 标题相似但作者/年份不一致 | 标注差异，建议用户确认 |
| 无法验证 | 所有渠道均无匹配 | 不入池，不引用，建议替换 |

## 教育学论文常用方法学术语

### 研究设计类型
- 实验研究：真实验、准实验、单组前后测
- 准实验设计：非等组前后测、时间序列
- 调查研究：横断面、纵向追踪
- 个案研究：单案例、多案例
- 行动研究
- 混合方法：并行设计、顺序解释性设计、顺序探究性设计

### 抽样方法
- 随机抽样：简单随机、分层随机、整群随机
- 非随机抽样：便利抽样、目的性抽样、滚雪球抽样

### 数据收集工具
- 问卷/量表（需报告信效度：Cronbach's α、内容效度指数CVI）
- 访谈（半结构化、深度访谈、焦点小组）
- 观察（参与式、非参与式、结构化）
- 测验/考试
- 档案/文件分析

### 数据分析
- 描述性统计：均值(M)、标准差(SD)、频数、百分比
- 推断统计：t检验、方差分析(ANOVA)、卡方检验、回归分析
- 效应量：Cohen's d、η²、Cramer's V
- 质性分析：主题分析、扎根理论、内容分析

## GRADE 证据质量评级速查

| 等级 | 符号 | 含义 |
|------|------|------|
| 高 | ⊕⊕⊕⊕ | 真实效应接近效应估计值 |
| 中 | ⊕⊕⊕⊖ | 真实效应可能接近效应估计值，但存在差异可能 |
| 低 | ⊕⊕⊖⊖ | 真实效应可能与效应估计值明显不同 |
| 极低 | ⊕⊖⊖⊖ | 真实效应很可能与效应估计值明显不同 |

### 降级因素
1. 偏倚风险（-1～-2）
2. 不一致性（-1～-2）
3. 间接性（-1～-2）
4. 不精确性（-1～-2）
5. 出版偏倚（-1）

### 升级因素（仅观察性研究）
1. 大效应量（+1～+2）
2. 剂量-反应关系（+1）
3. 残留混杂可能降低效应（+1）
